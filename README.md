# 📝 Task Manager v2

A modern Task Manager web application built using **HTML**, **CSS**, and **JavaScript**. It helps users manage daily tasks with features like task creation, editing, searching, sorting, priority levels, due dates, dark mode, and persistent storage using Local Storage.

---

## 🚀 Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ✅ Mark tasks as completed
- 🔍 Search tasks
- 📊 Task counter (Total, Completed, Pending)
- 🔥 Priority levels (High, Medium, Low)
- 📅 Due date for each task
- 🌙 Dark Mode
- 💾 Data saved using Local Storage
- 📱 Responsive Design
- ✨ Smooth UI animations

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
